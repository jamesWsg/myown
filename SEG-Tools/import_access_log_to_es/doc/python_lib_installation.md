# Python依赖库安装 

## 使用root用户上传


```
# ls
urllib3-1.22.tar.gz
elasticsearch-6.2.0.tar.gz
configparser-3.5.0.tar.gz
# 

```

## 安装urllib3

执行如下命令

```
#tar xvf urllib3-1.22.tar.gz
#cd urllib3-1.22 
#python setup.py install
```

如果没有出错，说明安装完成
